## These are the features of this app:
+ [x] Draw by Pen
+ [x] Draw a line
+ [x] Reset the workspace
+ [x] Change the size of the point
+ [x] Change the color of the point
+ [x] Dark / Light theme Toggle
+ [x] Resize the workspace
